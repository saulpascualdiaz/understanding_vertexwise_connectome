# Understanding Mediation

This repository contains an interactive tool for visualizing and understanding vertexwise structural connectivity matrices.

## Live Website
The tool is hosted and accessible at:
**[https://saulpascualdiaz.github.io/understanding_mediation/](https://saulpascualdiaz.github.io/understanding_vertexwise_connectome/)**

---

## Overview
*Understanding vertex-wise structural connectomes* — an interactive single-file HTML tutorial that explains how a ~244,000 × 244,000 sparse cortical-vertex connectome is built from a tractogram, SIFT2 weights, FreeSurfer surfaces, and a DWI b0, with a built-in script generator that emits a ready-to-run bash + Python pipeline for your own data.
